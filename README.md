# MediSync — Plataforma de Gestión Médica Digital

Proyecto final del curso **Diseño y Arquitectura de Software** — Universidad Tecmilenio.

MediSync es una plataforma web para la gestión de pacientes, citas médicas, expedientes clínicos y notificaciones, diseñada para un consultorio médico administrado por el Dr. Ramírez.

---

| Capa | Tecnología |

| Backend | Node.js + Express |

## Módulos del sistema

1. Gestión de Pacientes
2. Agenda de Citas
3. Notificaciones
4. Expediente Médico Digital
5. Dashboard Administrativo
6. Catálogo de Médicos

## Definition of Done

Una historia de usuario está **Done** cuando:
- El código está en el repositorio y pasó code review de al menos 1 compañero
- Cumple todos los criterios de aceptación y fue probada manualmente
- Funciona en Chrome y Firefox sin errores en consola
- Está integrada sin romper funcionalidad existente
- Está desplegada en staging y validada por el SM
- El ticket en Jira tiene evidencia (captura o video)

> Si el Dr. Ramírez la pidiera en la demo ahora mismo, ¿podría mostrarse sin preparación? Si sí, está Done.
